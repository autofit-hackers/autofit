# posefit-streamlit-example

## Prerequisite
* python 3.8.13
```
$ pip install -r requirements.txt
```

## Demo (Recommended)
```
$ streamlit run streamlit_app.py
```
> :warning: ブラウザからローカルに立てたサーバに接続する際に、カメラの使用を許可しないと動作しない
